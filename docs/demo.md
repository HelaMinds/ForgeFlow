# Demo

## Example Input

> I want to launch a small online tutoring platform for high school students in my city.

Idea type: **Side hustle**

## Flow

1. Submit idea on home page
2. Answer 3–5 clarifying questions on `/clarify`
3. View personalized roadmap on `/result`

## Expected Output

- Your answers reflected in the plan
- Clarified goals and constraints
- Phased execution roadmap
- Risk analysis with mitigations
- One concrete first action to start immediately
- Disclaimer: plan is decision support — you choose whether to pursue it

## Try It

1. Start the backend: `npm run dev:backend`
2. Start the frontend: `npm run dev:frontend`
3. Open the frontend URL (e.g. `http://localhost:3000`) and submit an idea
